How to Play ?

1. First Player (Playing X) is allowed to Play in any Grid (All Grids are Active) 2. The Player Chooses the Position (Square) on that Grid - and the same Position of the Square Choosen by First Player is the active Grid on which the second Player Plays - Second Player has to play on that Active Grid - example: 1. Player X Chooses Top Left of a Square in the Center Grid 2. Since the Choosen Square is Top Left, the next Active Grid is Top Left 3. Player O plays the Top Left Grid and Chooses a Square Center 4. Choosen Square is Center so, Center Grid Becomes Active 5.Player X chooses a Square in Center Grid 3. A Grid is captured by the Player if he wins in that Grid. The same Grid will never become Active Grid. - If the same Position of a Square is Played - Other Grids Become Active - example: 1. Player X wins Top Left Grid 2. If Player O chooses the Top Left Square in any Active Grid, Player X will be allowed play any other Grid 3. Player X will be able to play any Grid except the Top Left Grid (or any Grid won or that which becomes draw) 4. A Grid Becomes Draw if there are no Squares to Play 5. The final Game is won if a Player Wins 3 Grids in the Order of a Normal Tic-Tac-Toe

1. Players can only Play in an Active Grid (Black Colored) 2. Players can't play in Grid won before, or that becomes draw 3.
